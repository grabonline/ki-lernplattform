# KI-Lernplattform fuer Kinder

**Ein KI-Tutor der mit echten Klassenarbeiten trainiert wird — nicht generisches ChatGPT, sondern einer der genau weiss was DEIN Lehrer fragt.**

## Vision

Kinder laden ihre Klassenarbeiten, Arbeitsblaetter und Schulbuecher hoch. Die KI lernt daraus und wird zum persoenlichen Nachhilfe-Lehrer der:

- Genau weiss was in der naechsten Klassenarbeit drankommt
- Den Stoff erklaert wie ein geduldiger Freund
- Uebungsaufgaben erstellt die zum Niveau passen
- Schwaechen erkennt und gezielt trainiert
- 24/7 verfuegbar ist — nicht nur Dienstags 16 Uhr

## Wie es funktioniert

```
Kind laedt Klassenarbeit hoch (Foto/PDF)
    ↓
KI analysiert: Themen, Schwierigkeitsgrad, Fehler-Muster
    ↓
Wissen wird in persoenliche Knowledge-Base gespeichert (RAG)
    ↓
Kind fragt: "Erklaer mir Bruchrechnung"
    ↓
KI antwortet basierend auf ECHTEM Schulstoff — nicht Wikipedia
    ↓
KI generiert Uebungsaufgaben im Stil des Lehrers
    ↓
Kind uebt → KI gibt Feedback → Wiederholen bis es sitzt
```

## Technologie

- **Notebook LLM** — Kinder koennen Wissen reinkanalisieren
- **RAG (Retrieval Augmented Generation)** — persoenliche Wissens-Datenbank pro Kind
- **Claude / GPT** — erklaert, generiert Aufgaben, gibt Feedback
- **OCR** — Klassenarbeiten als Foto hochladen

## Warum das besser ist als Nachhilfe

| | Nachhilfe | KI-Lernplattform |
|---|---|---|
| Verfuegbarkeit | 1-2x pro Woche | 24/7 |
| Kosten | 30-50 EUR/Stunde | Flatrate |
| Personalisierung | Tutor kennt Kind ein bisschen | KI kennt JEDE Klassenarbeit |
| Geduld | Mensch wird muede | Unendlich geduldig |
| Schulstoff-Naehe | Tutor raet was drankommt | KI hat die echten Arbeiten |
